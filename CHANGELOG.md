# Changelog

## v1.000 (2026-04-01)

Initial release.

**Upstream DINish:** playbeing/dinish @ `HEAD` (v4.006)

### Changes from upstream DINish
- Rename: DINish → Dinsy
- UPM: 1024 → 1000 (exact 120 px line height at 100 px font-size)
- Line metrics: match DINsical (hhea 830/−170/200, sTypo 750/−250/200, win 850/350)
- Glyph scale: ×0.985 to match DINsical visual glyph size
- Drop Condensed and Expanded variants (regular width only)
- Variable font: `wght` + `slnt` axes (no `wdth`)
