# Dinsical

Dinsical is a variable sans-serif typeface derived from
[DINish](https://github.com/playbeing/dinish) (itself a clean-room
recreation of DIN 1451 / Altinn-DIN / D-DIN). It offers a three-axis
variable font with weight, width, and slant axes, plus a full set of
static instances.

## Variable font axes

| Axis   | Tag    | Range            | Instances |
|--------|--------|------------------|-----------|
| Weight | `wght` | 300 – 900        | Light, Regular, Medium, SemiBold, Bold, Heavy, Black |
| Width  | `wdth` | 75 – 125         | Condensed (75), Regular (100), Expanded (125) |
| Slant  | `slnt` | 0 – −10          | Upright (0), Italic (−10) |

## Font metrics

| Metric | Value |
|---|---|
| UPM | 1000 |
| hhea ascender / descender / lineGap | 830 / −170 / 200 |
| sTypo ascender / descender / lineGap | 830 / −170 / 200 |
| OS/2 WinAscent / WinDescent | 850 / 350 |
| **Line height @ 100 px** | **120.000 px** |

## OpenType features

| Feature | Description |
|---|---|
| `tnum` | Tabular figures |
| `onum` | Old-style numerals |
| `frac` | Diagonal fractions |
| `ss01` | Stylistic set 1 (Dutch IJ) |
| `ss02` | Alternate `a` (activated automatically in italics) |

## Language support

243 Latin-based languages; full European Latin coverage; Cyrillic
(Russian, Bulgarian, Serbian); old-style and tabular numerals.
